01 Game Overview

Perseus' Quest is a Greek-themed turn-based RPG where the player navigates through three areas: a town where they can buy supplies, a temple to communicate with the Greek Goddess Athena, and a dungeon where the player is tasked with slaying monsters and finding treasures.

02 Game Goal

The goal of the game is to defeat the final boss, Medusa, and gift her head to Athena at the Temple.

✓ Win Condition
Gift Medusa's head to Athena.
✗ Lose Condition
Health reaches zero.

04 Game Mechanics & Logic

Main Actions:

  • Move through the areas
  • Fight monsters
  • Buy items
  • Talk to Athena

Variables:

Variable Name Starting Value What It Does
Gold10Tracks how much currency the player currently has
health25How many hits the player can take
Stamina100How many special attacks the player can make
Exp.0Tracks how much more the player needs to level up
Level1Tracks the strength of the player; higher levels mean more health and stamina

If / Then Rules:

  • If the player collects coins → add 1–3 gold
  • If the player is hit by an attack → lose 1–2 health
  • If health equals 0 → the game ends (lose)
  • If the player kills a monster → gain 10–20 exp.
  • If the player gives Medusa's head to Athena → the player wins

05 Game Loop

These steps repeat over and over while the game is running:

  1. Check for player input (did the player press a key or button?)
  2. Move the menu selector based on the input
  3. Check for encounters (is the player engaged in a battle?)
  4. Update gold, exp., or health based on the enemy
  5. Check win or loss conditions
  6. Repeat until the game ends

06 Playtesting & Changes

Changes made after testing the game with real players:

⚠ Problem Found

Players said the game was too easy.

✓ Changes Made

Added more obstacles and reduced starting time from 80 to 60 seconds.
